# 山西省人才网数据爬取项目

## 任务目标

1. 掌握 requests 发送网页请求的方法
2. 掌握正则解析网页内容的方法
3. 掌握 mysql 存储数据的方法
4. 掌握多页爬取的方法

## 任务要求

> [山西人才网](https://www.sjrc.com.cn/) 是山西地区专业的人才网之一，在原有的山西人才网的基础上升级改造成山西第一家视频招聘网，
> 
> 其品牌和服务已被广大个人求职者和企业人力资源部门相关负责人普遍认可已成为太原地区大型. 专业的人才网站之一。本项目以山西人才网为目标，主要获取计算机领域软件开岗位的招聘信息。

1. 通过谷歌浏览器的开发者模式对网页进行分析
2. 通过 requests 库向网页发送请求并获得相应
3. 通过正则表达式，将网页内容进行解析，提取的内容包括：公司名称. 招聘岗位. 地址. 薪资待遇，发布日期
4. 从页面中解析出网址或分析网页链接，实现多页的爬取
5. 将爬取的非格式化的数据保存为 mysql 数据库中
